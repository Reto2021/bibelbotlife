Fonts:
- Display / Headings: **Instrument Serif** (via `font-display` oder h1/h2/h3). Wichtige Wörter kursiv im primary-Ton (`italic text-primary`).
- Body: **Inter Tight** (default `font-sans`). Feature-Settings cv11/ss01/ss03 aktiv.
- Serif prose (Bible-Verse etc.): weiterhin Lora (`font-serif`).

Tokens:
- `--gradient-hero` = radialer Warmton oben.
- `--gradient-glow` = weicher Kerzenlicht-Radial. Utility: `bg-glow` + `animate-candle` (6s breathe).
- `--shadow-warm` / `--shadow-elegant` als Utilities `shadow-warm` / `shadow-elegant`.
- `.text-balance` Utility für Headline-Umbruch.
